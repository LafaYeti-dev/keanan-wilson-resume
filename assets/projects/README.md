# Public Project Assets

Add only sanitized, portfolio-safe assets approved for public display.

Suggested folders:

- `skin-element/`
- `jennifer-carly/`
- `sound-united/`
- `hk-strategies/`

## Do not commit

- supplier or retailer spreadsheets
- Shopify exports or private admin screenshots
- credentials or private URLs
- client contracts or internal documents
- confidential analytics dashboards
- assets without public-display rights

For The Skin Element, use screenshots from the archived unreleased Shopify build and label them accordingly. Do not use the current live public site as evidence of that build.
